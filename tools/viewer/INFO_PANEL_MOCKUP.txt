with the 'I' key or via View → Model Information menu.

Key Features:
=============

1. TOGGLE: Press 'I' to show/hide
2. AUTOMATIC UPDATES: Panel refreshes when loading new files
3. COMPREHENSIVE: Shows file info, geometry stats, extensions, and objects
4. CLEAN DESIGN: Organized sections with headers and clear formatting
5. MINIMAL: Only shows relevant information, no clutter

Implementation:
==============

- Panel state tracked in ViewerState.info_panel
- Rendering via render_model_info_panel() function
- Uses kiss3d text rendering
- Positioned at (10, 40) pixels from top-left
- Light gray text on semi-transparent background
- Yellow headers for section titles
